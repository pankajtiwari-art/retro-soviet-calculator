A web-based clone of the vintage Soviet "Электроника С3-22" calculator, featuring an authentic retro UI and mechanical sound effects.

**🔗 [Play with the Live Demo Here!](https://pankajtiwari-art.github.io/retro-soviet-calculator/)**

---

## 🌟 Features

* **Authentic Retro UI:** Designed to mimic the classic aesthetic of the Soviet Elektronika C3-22 calculator, complete with custom fonts and stylized 3D buttons.
* **Mechanical Sound Effects:** Uses the browser's native **Web Audio API** to procedurally generate a nostalgic, 8-bit mechanical "click" sound on every keypress—no external audio files required.
* **Thematic Error Handling:** Dividing by zero triggers an authentic Russian error state: **`ОШИБКА`**, keeping the USSR theme intact.
* **Lightweight & Fast:** Built entirely with pure HTML, CSS, and Vanilla JavaScript, stripping away unnecessary frameworks for maximum performance.
* **Input Limitations:** Prevents overflow by restricting input length to 10 digits, ensuring the UI remains clean and unbroken.

## 🛠️ Tech Stack

* **HTML5:** Semantic structure.
* **CSS3:** Custom styling, gradients, shadows, and 3D perspectives for the vintage hardware look.
* **Vanilla JavaScript:** Core logic, state management, and DOM manipulation.
* **Web Audio API:** Oscillator-based sound generation for mechanical clicks.

## 🚀 How to Run Locally

Since this project uses pure HTML, CSS, and JS, setting it up is incredibly simple.

1. **Clone the repository:**
    git clone https://github.com/pankajtiwari-art/retro-soviet-calculator.git

2. **Navigate to the project directory:**
    cd retro-soviet-calculator

3. **Open the application:**
    Simply double-click on `index.html` to open it in your default web browser. No local server setup is required!
